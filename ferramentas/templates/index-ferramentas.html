Ajuda Comando

Algoritmo

SELECT <-- sempre tem que ter

COLUNAS <-- nome das colunas ou asterisco para pegar todas

FROM <-- sempre tem que ter

MATERIAIS <-- nome da tabela de produtos

WHERE <-- caso você queira filtrar valores

Exemplos

SELECT * FROM MATERIAIS

SELECT * FROM MATERIAIS WHERE VLR_CUSTO > 50

SELECT * FROM MATERIAIS WHERE VLR_CUSTO > 50 AND INATIVO = 'N'

SELECT CODID, COD_INTERNO, DESCRICAO FROM MATERIAIS

Instruções

1. Tirar pedidos

Antes de fazer o balanço de produtos, deve-se obrigatoriamente fazer a emissão de todos os pedidos.

2. Inativar robô (sincronização de pedidos)

Inative o robô que a todo momento faz a busca de novos pedidos no marketplace e debita o estoque. Isso pode atrapalhar o balanço! Para fazer isso você terá que ligar no Ambar X para fazer a solicitação de inativação. Contato: (011) 2087-6767

Despacho de produtos

Deve-se fazer o despacho no mesmo dia de todos os pedidos gerado. Não pode haver pedidos no sistema e nem na expedição, pois poderá afetar a contagem dos produtos.

4. Verificação de produtos

Antes de fazer o balanço de produtos, faça a verificação se existe algum produto no controle de qualidade ou no atendimento, pois os produtos que não estão no estoque, também devem ser contabilizados.

5. Baixe a planilha e faça o balanço


6. Importe a planilha

Deixe em BRANCO/VAZIO os produtos que você não fez o balanço.

Não exclua a coluna CODID e ESTOQUE_REAL

7. Ative o robô (sincronização de pedidos)

Ligue novamente para o Aton para fazer a ativação da sincronização de pedidos.
